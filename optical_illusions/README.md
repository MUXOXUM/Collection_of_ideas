# Optical Illusions

Минималистичная веб-страница с подборкой оптических иллюзий.

При открытии пользователь сразу видит иллюзию в центре экрана без кнопок, меню и подписей. Переключение между изображениями выполняется только с клавиатуры:

- `←` и `→`
- `A` и `D`

Страница сначала пытается автоматически прочитать содержимое папки `images` и собрать галерею без ручного перечисления файлов.

Если сервер не отдаёт список файлов в каталоге, используется резервный список изображений, зашитый в `index.html`.
