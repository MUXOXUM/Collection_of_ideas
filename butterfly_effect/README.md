Интерактивная 3D-визуализация системы Лоренца — математической модели детерминированной хаотической системы.

### Основная концепция

Пользователю демонстрируется эффект бабочки — концепция из теории хаоса, согласно которой незначительное изменение начальных условий может привести к кардинально разным результатам в развитии системы.

### Реализация

Интерфейс позволяет задать процентную разницу между начальными значениями `x` двух идентичных систем Лоренца (по умолчанию `0.001%`). Две траектории визуализируются одновременно:

- Первая — голубая
- Вторая — коралловая

Несмотря на то, что системы стартуют с почти одинаковыми параметрами, с течением времени их траектории начинают заметно расходиться, что наглядно иллюстрирует чувствительность хаотических систем к начальным условиям.

### Взаимодействие

Пользователь может вращать, масштабировать и панорамировать трёхмерную сцену для детального изучения расхождения траекторий, а также изменять величину начального отклонения в реальном времени.
